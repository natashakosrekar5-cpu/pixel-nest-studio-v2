01 / Strategic Opportunity

Eliminating Catalog Friction in Wellness Discovery

Wellness products require clear ingredient transparency and sensory storytelling. Customers wanted to filter items by specific benefits (sleep aid, stress relief, skin radiance) rather than navigating flat category lists. The opportunity was to design an interactive filtering matrix that matches products to user intent seamlessly.

02 / Design & Technical Constraints

Instant Client-Side Filtering Without Page Reloads

The entire discovery engine had to function dynamically on the client side without relying on complex database queries or external API latencies. Filtering speed needed to be instant (<50ms response) while maintaining smooth CSS layout transitions.

Product Architecture

What Needed To Change

Transitioning from static catalog pages to an active intent-driven product engine.

Legacy Problem 01

Static Unfilterable Grid

Products were displayed in a flat, unorganized list requiring users to scroll endlessly to find relevant wellness items.

Legacy Problem 02

Hidden Product Ingredients

Key wellness benefits, botanical origins, and organic certifications were buried inside long text blocks.

Legacy Problem 03

Multi-Step Checkout Friction

Adding items required navigating away from the current catalog view, causing high drop-off rates on mobile devices.

Methodology

Design Process Timeline

A thorough 6-stage engineering sprint focused on discovery UX and conversion efficiency.

PHASE 01

User Behavior Analysis

Mapped user search patterns to identify top filtering dimensions: Botanical Type, Use Case, and Price Range.

PHASE 02

Taxonomy Mapping

Structured 24+ SKUs into clean, intuitive categories with tag-based metadata for instant cross-filtering.

PHASE 03

Quick-Filter Wireframes

Designed a horizontal pill filter system positioned prominently at the top of the mobile viewport.

PHASE 04

Card UI Design

Crafted compact product cards showcasing ingredient badges, price points, and instant WhatsApp buy buttons.

PHASE 05

Micro-Interactions

Implemented smooth hover elevation, quick add animations, and instant category tab transitions.

PHASE 06

Performance Tuning

Optimized asset loading and DOM node counts to achieve 60fps scrolling performance on all mobile devices.
